# Smart-Library-Management-System-Java-JDBC
Advanced console-based Smart Library Management System using Java, JDBC, MySQL, OOP, Collections, Streams, Multithreading, and Design Patterns.

# Smart Library Management System

## Technologies Used
- Java 17
- JDBC
- MySQL
- Eclipse IDE

## Features
- User Authentication
- Book Management
- Borrow & Return System
- Fine Calculation
- Notifications
- Streams API
- Multithreading

## Design Patterns
- Singleton
- DAO
- Factory
- Observer

## Database
MySQL + JDBC

## Concepts Used
- OOP
- Collections
- Exception Handling
- File Handling
- Streams & Lambda
- Multithreading

## How to Run
1. Create MySQL database
2. Run SQL scripts
3. Add JDBC driver
4. Run LibraryManagementApp.java
